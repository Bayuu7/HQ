# System Architecture

This document describes the system architecture:

- **Core AI**: Models (Diffusion, Transformer, NeRF), Pipelines, Builders, Textures, Animation, Utils.
- **Services**: API Gateway, Job Orchestrator, Inference Worker, Storage Proxy.
- **UI**: React components (PromptInput, ModelViewer, TexturePreview, AnimationPlayer).
- **Clients**: Python + JS SDKs.
- **Infra**: Docker, Helm, Terraform, CI/CD pipelines.
- **Tests**: Unit, Integration, Performance, Security.
